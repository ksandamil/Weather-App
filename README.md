🌤️ Sky Cast – Weather Tracking Web Application
--------------------------------------------------------------------

Sky Cast is a modern, interactive weather tracking web application that provides real-time weather updates with a visually appealing glassmorphism user interface. The application enables users to search for cities and view accurate weather information, including temperature, weather conditions, wind speed, humidity, and local time, delivered through dynamic icons and day/night theme support.

The app features a 5-day weather forecast, allowing users to plan ahead based on upcoming weather conditions. Users can save favorite cities for quick access, while animated loaders enhance the overall user experience by providing smooth visual feedback during data fetching. Sky Cast integrates with the OpenWeatherMap API to ensure reliable and up-to-date weather data and is built using HTML, CSS, and JavaScript, focusing on responsiveness, performance, and clean UI design.
